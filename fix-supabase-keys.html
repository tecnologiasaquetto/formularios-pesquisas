🔧 Corrigir Chaves Supabase

Esta ferramenta vai validar e corrigir suas chaves do Supabase.

📋 Passo 1: Cole suas chaves do Supabase Dashboard

📊 Resultado da Validação

🔗 Teste de Conexão

📝 Passo 2: Gerar .env.local

Depois de validar, copie as chaves corretas:

REACT_APP_SUPABASE_URL=
REACT_APP_SUPABASE_ANON_KEY=
REACT_APP_SUPABASE_SERVICE_ROLE_KEY=

🔍 Como encontrar as chaves corretas

  1. Acesse: supabase.com
  2. Entre com sua conta
  3. Vá para: Project Settings → API
  4. Copie: Project URL e anon key
  5. Importante: Copie a chave COMPLETA (geralmente 200+ caracteres)

⚠️ Sinais de chave incorreta:

  • ❌ Erro 401: "Invalid API key"
  • ❌ Chave muito curta (menos de 200 caracteres)
  • ❌ Chave truncada no meio
