🔧 Configurar Supabase

Siga os passos abaixo para configurar sua conexão com o Supabase.

📋 Passo 1: Encontre suas credenciais

  1. Acesse supabase.com
  2. Faça login com sua conta
  3. Vá para Project Settings > API
  4. Copie a Project URL e anon key

📝 Passo 2: Identifique a URL correta

A URL do seu projeto segue este padrão:

https://[PROJECT_ID].supabase.co

Onde [PROJECT_ID] é o ID único do seu projeto.

Exemplo: https://abcdefghijklmnopqrstuv.supabase.co

⚙️ Passo 3: Atualize seu .env.local

Edite o arquivo .env.local na raiz do projeto:

REACT_APP_SUPABASE_URL=https://[PROJECT_ID].supabase.co
REACT_APP_SUPABASE_ANON_KEY=[SUA_ANON_KEY]
REACT_APP_SUPABASE_SERVICE_ROLE_KEY=[SUA_SERVICE_KEY]

Importante: Substitua [PROJECT_ID], [SUA_ANON_KEY] e [SUA_SERVICE_KEY] pelos valores reais.

🚀 Passo 4: Teste a conexão

Depois de configurar, acesse:

http://localhost:8080/test-supabase

Deve aparecer ✅ Conectado ao Supabase!

🔍 Como encontrar o Project ID

Se não encontrar o Project ID no Dashboard:

  1. Verifique o URL do navegador ao acessar o Supabase
  2. O Project ID está na URL após https:// e antes de .supabase.co
  3. Exemplo: https://abcd1234.supabase.co → Project ID é abcd1234

🛠️ Solução de problemas comuns

❌ Erro "Multiple GoTrueClient instances"

Feche outras abas do navegador e recarregue a página.

❌ Erro "CORS policy"

Configure as CORS policies no Supabase Dashboard.

❌ Erro "Connection refused"

Verifique se o projeto Supabase está ativo.
